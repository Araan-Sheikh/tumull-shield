# Contributing

Hey, thanks for considering contributing!

## Setup

1. Fork & clone: `git clone https://github.com/YOUR_USERNAME/tumull-shield.git`
2. Install: `npm install`
3. Branch: `git checkout -b my-feature`

## Dev commands

```bash
npm run build        # build with tsup
npm test             # run tests
npm run test:watch   # tests in watch mode
npm run test:coverage
npm run format       # prettier
```

## PRs

- Write tests for new stuff
- Make sure `npm test` passes
- One feature/fix per PR please
- Update docs if relevant

## Issues

Open a GitHub issue. Include a minimal repro if you can, plus your Node version and what framework you're using.
